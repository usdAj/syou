第2章 まとめ

プログラムの骨格（おまじない）

public class ClassName※{
    public static void main(String[] args){
        命令文等
    }
}
※ClassNameは任意の名前に変更してください。

出力（改行無し）

System.out.print("Hello World");

出力（改行有り）

System.out.println("Hello World");
あるいは

System.out.print("Hello World\n");
ただし、太字の\nはエスケープ・シーケンス※です。

エスケープ・シーケンスは文字列の中で特別な意味（動作）を持っています。

コンパイル・実行

コンパイル > javac ClassName.java
実行 > java ClassName

注意：ソースコードを変更した場合には、実行する前に必ずコンパイルを行うこと。

コラム

※エスケープ・シーケンス

バックスペース ¥b
バックスラッシュ ¥¥
改行 ¥n
一重引用符（シングルクォート） ¥'
二重引用符（ダブルクォート） ¥"
タブ ¥t
