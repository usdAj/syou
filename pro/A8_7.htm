8.7 コマンドプロンプトからの入力1

プログラム

プログラムの説明

コマンドプロンプトからの入力1

プログラムA8_5をコンパイル・実行してください。

実行する時には

> java A8_5

> java A8_5 r

などとしてください。

実行結果はコマンドプロンプトで与えた「javaクラス名」以降の文字列がない場合にはエラーが、文字列がある場合にはその文字列が出力されたことを確認してください。

下記の手順に従って実行してみましょう！

エディタにプログラム入力

プログラムA8_5をエディタに入力・保存します。ファイル名はA8_5.javaです。
コンパイル・実行

A8_5.javaをコンパイル・実行します。コンパイルでエラーが出力されている間はプログラムを変更しコンパイルからやり直してください。
結果

実行結果はコマンドプロンプトで与えた「javaクラス名」以降の文字列がない場合にはエラーが、文字列がある場合にはその文字列が出力されたことを確認してください。
