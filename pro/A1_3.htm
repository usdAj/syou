1.3 プログラムの入力方法

プログラムA1_1

public class A1_1{
    public static void main(String[] args){
        System.out.print("Hello World");
    }
}

入力手順１

public class A1_1{
}

入力手順2

public class A1_1{
    public static void main()
}

入力手順3

public class A1_1{
    public static void main(String[] args)
}

入力手順4

public class A1_1{
    public static void main(String[] args){
    }
}

入力手順5

public class A1_1{
    public static void main(String[] args){
        System.out.print();
    }
}

入力手順6

public class A1_1{
    public static void main(String[] args){
        System.out.print("Hello World");
    }
}

説明

プログラムの入力方法

プログラムを入力するときには波括弧（ブロック）や括弧を意識して入力を行ってください。
