2.3 文字列の出力（複数の出力命令文）

プログラムA2_3

説明

文字列の出力（複数の出力命令文）

では、System.out.printを3行ならべると、出力がどうなるかを実際に体験しましょう。このプログラムを入力・保存します。ファイル名はA2_3.javaとしてください。

コンパイル

コマンドプロンプトでjavac A2_3.javaと入力しリターンキーを押してください。。

エラー

コンパイルでエラーが出力されている間はプログラムを変更しコンパイルから、やり直してください
実行と結果

コマンドプロンプトでjava A2_3と入力しリターンキーを押してください。この結果、各自の画面にHello WorldがSystem.out.printを並べた数だけ横につながって出力されたことを確認してください。

操作手順

エディタにプログラム入力

プログラムA2_3をエディタに入力・保存します。ファイル名はA2_3.javaです。
コンパイル

javac A2_3.javaと入力しリターンキーを押します。コンパイルでエラーが出力されている間はプログラムを変更しコンパイルからやり直してください。
実行と結果

java A2_3と入力しリターンキーを押します。System.out.printを並べた数だけ横につながって出力されたことを確認します。
